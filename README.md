# EcoTexture-AI
Egypt generates approximately 21 million tonnes of municipal solid waste annually, yet only 4-5% is formally recycled. A major bottleneck is inaccurate automated waste sorting due to crushed, deformed, and reflective items that confuse traditional deep learning models. These models suffer from "texture blindness," relying heavily on global shapes and colors rather than surface material properties.

Additionally, there is a significant education gap — students and communities lack interactive tools to understand recycling and AI decision-making. 

EcoTexture AI addresses both technical and educational challenges by developing an explainable hybrid AI system that combines deep semantic features with local invariant texture descriptors (SIFT) using lightweight cross-modal attention. The project aims to improve sorting accuracy while turning the technology into a practical sustainability education tool for Egyptian schools.
